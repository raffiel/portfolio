Thesis Project

In this study, I conducted sentiment analysis of the 2024 Indonesian presidential candidates on the Twitter social media platform using the TF-IDF and naive Bayes methods. Twitter was chosen due to its widespread use among the Indonesian population, making it a strategic platform for observing public opinions regarding the 2024 presidential candidates. The subjects of sentiment analysis were declared presidential candidates, including Prabowo Subianto, Ganjar Pranowo, and Anies Baswedan.

The dataset of tweets used in this analysis covered the month of October 2022, collected through web crawling using the Twitter API. Subsequently, the tweet data underwent a preprocessing stage to clean out irrelevant elements. Following this, the data was labeled using an Indonesian-language lexicon to determine sentiment. The final results of the analysis can be found in the concluding section of this content.

Program Development Cycle

1. Planning

Plan the data collection from Twitter using the Twitter API, specify the presidential candidates for sentiment analysis, including Prabowo Subianto, Ganjar Pranowo, and Anies Baswedan, determine the time period for collecting tweets, which is October 2022.

2. Data Collection

Utilize the Twitter API to collect tweets from October 2022, which include the specified presidential candidates. Store the tweet data in a suitable format for further analysis

3. Preprocessing Data

Cleanse tweet data by removing unnecessary elements such as links, special characters, and the use of slang. Normalize text to address spelling and formatting differences. Eliminate duplicate data if present

4. Sentiment Labeling

Utilize an Indonesian language lexicon dictionary to assign sentiment labels (positive, negative, neutral) to each tweet. Adjust parameters to determine sentiment based on TF-IDF and naive Bayes scores

5. Sentiment Analysis

Calculate sentiment scores for each presidential candidate based on related tweets. Create visualizations of sentiment analysis results, such as graphs or diagrams, to present findings.

6. Evaluation and Further Processing

Review the analysis results and evaluate whether they align with the initial objectives. If necessary, carry out further processing, such as identifying specific sentiment patterns or trends.
